# NoSQL Injection Lab – TryHackMe Writeup

This repository contains my walkthrough for the NoSQL Injection Lab on TryHackMe.

The goal of this lab is to understand how NoSQL injection vulnerabilities occur and how attackers can exploit MongoDB queries to bypass authentication and retrieve sensitive information.

## Skills Practiced

- Understanding NoSQL databases
- MongoDB query manipulation
- Authentication bypass
- Operator based NoSQL injection
- Data extraction techniques

## Tools Used

- Web Browser
- Burp Suite (optional)
- Manual payload testing

## Concepts Learned

- MongoDB query structure
- NoSQL injection detection
- Boolean based injection
- MongoDB operators exploitation

## Disclaimer

All techniques demonstrated here were performed inside the TryHackMe lab environment for educational purposes only.
